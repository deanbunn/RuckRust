## Ruck Rust

My personal repo for learning Rust. 


## Project Commands
```bash
#Create New Project
cargo new "project_name"

#Build Project
cargo build

#Run Project
cargo run 

#Update Project
cargo update

```

## Install on Ubuntu Desktop
```bash
#Install Requirements
sudo apt install build-essential

#Install Rust
sudo apt install rustup

#Set Version
rustup default stable

#Verify Version
rustc --version

```
